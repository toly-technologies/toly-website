# Toly

Public website for Toly, an early-stage technology studio preparing thoughtful digital tools and experiences.

Live site: https://em-eshy.github.io/toly-website/

## Public Scope

This public repository is intentionally small. It contains only:

- `index.html` - the public landing page
- `README.md` - public repository documentation

Product pages, drafts, publishing scripts, private notes, and internal planning material are kept outside the public commit history.

## Current Position

Toly does not have public products yet. The website is deliberately simple while the work matures, with a reassuring tone that avoids overpromising and keeps public claims accurate.
