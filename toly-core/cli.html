CLI console

A private control room for publishing updates.

This page gives the publishing CLI its own UI. Run either backend locally, choose its endpoint, then publish a company update through the API instead of editing JSON by hand.

Open console View news feed

Backend options

Py

Standard-library API on http://127.0.0.1:8787.

Rs

No-dependency Rust API on http://127.0.0.1:8788.

API publish

Start a backend, then send an update.

Response

Waiting for backend activity.

Successful API publishes write to private/data/news.json.

Run commands

Choose one backend at a time.

These services are intentionally small. They are private scaffolding for the publishing workflow, not production authentication or hosting.

python backend/python/toly_cli_server.py

cd backend/rust
cargo run

The CLI console is private. Public visitors should only see approved company news.
